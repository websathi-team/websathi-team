# Contributing to WebSathi

Thank you for your interest in contributing to WebSathi!  
We welcome contributions from our team members and external collaborators.

---

## 📌 How to Contribute

### 1. Fork the Repository
Click the **Fork** button on the repository you want to contribute to.

### 2. Clone Your Fork
```bash
git clone https://github.com/your-username/repository-name.git

3. Create a New Branch
git checkout -b feature/your-feature-name

4. Make Your Changes

Write clean, readable, and maintainable code

Follow existing project structure and coding standards

Add comments where necessary

5. Commit Your Changes
git commit -m "Add: meaningful commit message"

6. Push to Your Fork
git push origin feature/your-feature-name

7. Open a Pull Request

Describe what you changed

Link related issues if applicable

Wait for review from the WebSathi team
